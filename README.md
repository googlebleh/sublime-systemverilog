# Sublime Text SystemVerilog
SystemVerilog Package for Sublime Text 2/3. Based on [Verilog plugin](https://github.com/pro711/sublime-verilog) by pro711.

Supports SystemVerilog files (*.sv, *.svp)

Features include
 - Syntax Hightlighting
 - Code snippets
